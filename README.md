## Core Capabilities

### Backend Engineering

Building APIs, services, integrations, and event-driven systems using practical, production-oriented patterns.

### Distributed Systems

Working with message queues, caches, databases, background jobs, synchronization, reconciliation, and reliability concerns.

### Blockchain Infrastructure

Exploring smart contracts, custody flows, withdrawal controls, on-chain/off-chain coordination, and EVM-based architecture.

### Cloud & DevOps

Working with Linux, AWS, Docker, reverse proxies, CI/CD, deployment automation, and operational tooling.

### AI Engineering

Experimenting with AI agents, MCP integrations, workflow automation, prompt-driven development, and multi-model tooling.
