## Blockchain Custody Flow

```mermaid
sequenceDiagram
    participant User
    participant Backend as Backend / Engine
    participant Recon as Reconciliation Service
    participant Contract as Funding Contract
    participant Chain as Blockchain

    User->>Backend: Withdrawal request
    Backend->>Backend: Validate balance and risk
    Backend->>Contract: Submit balance snapshot
    Recon->>Backend: Verify engine balance
    Recon->>Contract: Compare contract state
    Contract->>Chain: Execute controlled withdrawal
    Chain-->>User: Funds received
```
